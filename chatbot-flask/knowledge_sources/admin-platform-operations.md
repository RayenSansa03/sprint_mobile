# Admin Platform Operations

## Daily checks
- Authentication health and error rates.
- Queue backlog and task latency.
- AI model response consistency metrics.

## Weekly controls
- Role and permission review.
- Audit logs anomaly detection.
- Backup and restore spot verification.

## Incident readiness
- Maintain runbook for service degradation.
- Escalation path with communication templates.
- Post-incident review and corrective backlog.
